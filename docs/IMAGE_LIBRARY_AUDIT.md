## Launch Rule

An image is launch-safe only when at least one of these is true:

- It is real CreatorBridge-owned work.
- It is real creator/client submitted work with permission to display it.
- It is from a documented stock source such as Pexels or Unsplash and allowed for marketplace placeholder use.
- It is generated by us and is not presented as proof of a real creator portfolio.
- It has written approval recorded in this file or another repo document.

Unknown-source visible images must be replaced or documented before launch.

Reference scan used:

```bash
node - <<'NODE'
const fs=require('fs'); const cp=require('child_process');
const files=cp.execSync('rg --files src').toString().trim().split('\n').filter(Boolean);
const refs=new Map();
const re=/\/images\/creatorbridge\/[A-Za-z0-9_./-]+\.(?:jpg|jpeg|png|webp)/g;
for(const f of files){
  const s=fs.readFileSync(f,'utf8');
  for(const m of s.matchAll(re)){
    const img=m[0];
    if(!refs.has(img)) refs.set(img,new Set());
    refs.get(img).add(f);
  }
}
const rows=[...refs.entries()].sort((a,b)=>a[0].localeCompare(b[0]));
const buckets={handoff:0,backgrounds:0,topLevel:0,total:rows.length};
for(const [img] of rows){
  if(img.includes('/handoff/')) buckets.handoff++;
  else if(img.includes('/backgrounds/')) buckets.backgrounds++;
  else buckets.topLevel++;
}
console.log(JSON.stringify(buckets,null,2));
NODE
```
